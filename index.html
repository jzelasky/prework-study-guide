General Terminal Commands

  • ls shows a list of files and directories on the computer
  • pwd "print working directory", shows current directory
  • cd 'folder name' navigates into a folder
  • cd .. navigates backwards one directory at a time
  • mkdir 'directory name' creates a directory
  • touch 'file name' creates a file
  • code . opens VS code

Git and Github

  • Git is a version control system that lets you manage and keep track of each version of your project.
    • Git is accessed through the command line. Git commands always start with "git".
  • Github is a cloud-based hosting service that allows us to keep track of our projects inside repositories that ar e stored remotely. Github makes it easy to collborate with others.
  • Git commands:
    • git status checks what branch we are currently on
    • git checkout 'branch-name'switches branches
    • git checkout -b 'branch-name' creates a new branch and switches to it
    • git add -A stages changes made to a file to the working branch
    • git commit -m "text" commits changes to the repository with a message about the change (the -m flag indicates the message)
    • git pull origin main updates the local repository
    • git push origin 'branch-name' pushes changes to repository
  • After pushing changes, navigate to the working branch in the repository in your browser. Click on the "pull requests" tab then click the "new pull request" button. The base branch is the branch you want to merge into and the compare branch is you've been working on. Make sure you see a green "able to merge" message with a check mark. Click the "create pull request" button. Give the PR a title and short description about the changes you made. Click the "create pull request" button, then the "merge pull request" button and confirm.
  • Always update the local repository after making changes and before starting new changes.

Hypertext Markup Language (HTML)

  • HTML provides the structure of a webpage.
  • Elements are types of content on a webpage such as text, images, headers, and lists.
  • HTML attributes define additional information about an element. HTML attributes are always defined the beginning element, never the closing element.
  • The head element contains information, also known as metadata, about the webpage.
  • The body element represents the visible content shown to the user.

Cascading Style Sheets (CSS)

  • CSS is used to define how webpage content is presented such as colors, aesthetics, visual layout, and organization.
  • There are ways to use CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
    • Inline CSS is used to style an element directly in the HTML file. Inline styles are useful for quick and permanent changes but are less flexible than external and internal style sheets.
    • Internal style sheets contain CSS rules for the webpage in the head section of the HTML file.
    • External style sheets are separate files that contain CSS rules that are then linked to the HTML file. This is the most common method of using CSS because it's easier to make design changes. The link element is placed in the head of the HTML file.
  • A CSS rule contains two parts: a selector and a declaration block.
    • A selector determines what part of the webpage you want the CSS rule to apply to.
    • A declaration block is made up of one or more declarations. Each declaration has two parts: the CSS property we want to apply and the value of the property.
  • A class attribute is a type of selector that allows us to choose what which HTML elements we want the CSS rule to apply to.
  • A margin property indicates how much space we want around the outside of an element.
  • A padding property indicates how much space we want around the content inside an element.
  • A border property places a border around our element and indicates how thick and what color it should be.
  • A box-shadow property adds shadow effects around an element's frame.

JavaScript (JS)

  • JavaScript adds user interactivity to a webpage.
  • Without JavaScript a webpage is known as static.
  • A variable (var) is a named container that allows us to store data in our code. Variable names must be unique because it is how we will reference the information stored in the variable in our code.
  • '=' is an assignment operator used to assign the specific data that we want stored in our vairable.
  • A string is a list of variables surrounded by quotation marks.
  • Every line of JavaScript ends with a semicolon.
  • The console is a test environment that developers use to check their code.
  • A method is a set of instructions that can be executed by a computer.
    • The 'console.log()' method outputs a message to the web console.
  • .js files are linked in the body of a HTML file (as opposed to .css files which are linked in the head).
  • Chrome DevTools is a type of web console used in Google Chome. To open it, either right click and click 'inspect' or use Command+Option+I on MacOS or Control+Shift+I on Windows.
  • Method is the order in which a computer executes code in a script. In JavaScript, code runs from the top of the page to the bottom.
  • Conditional statements and loops are used to control the order in which code is executed, or if it's even executed at all.
    • If statments are a way of introduing decision making into our code. The code will only execute if the if statement meets a condition or is true.
    • A value is considered falsy if it is one of the following values: 0, -0, On, "", null, undefined, Nan, or a Boolean data type with the value of false.
    • A value is considered truthy if it is not falsy.
    • Else if is used if there is more than one condition.
  • '===' is the strict equality operator, it is used to check to see if two values are equal.
  • An array is a single variable that is used to hold a group of related data.
  • An index is a unique number used to tell the location of an item inside an array. The first item is assigned 0 and it counts up from there.
  • A for loop uses the predictable pattern of indices in an array to perform a task on al lthe items in an array.
    • A for loops contains three statements:
    • The first statement determines a starting point for our loop. If we want to start at the beginning then we declare a variable x and give it a value of 0.
    • The second statement is a condition that determines when the loop to end.
    • The third statement is what allow the array to iterate over each item. We usually want the code to loop for every item so the index will increase by 1 each time. The code x++ is a shortcut way of writing x+1.
    • An example of a for loop: for(var x=0; x<'var name'.length; x++) {console.log('var name'[x]);}
  • A function is a set of isntructions that tells the computer how to perform a certain task. A function's name is always followed by parentheses.
